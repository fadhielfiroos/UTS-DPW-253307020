Deskripsi Proyek: Portal PPDB Sekolah Rakyat Nusantara
Proyek ini adalah sebuah Website Sistem Informasi Pendaftaran Peserta Didik Baru (PPDB) untuk Sekolah Rakyat Nusantara. 
Website ini dirancang untuk memudahkan calon siswa dalam mengakses informasi pendaftaran dan melakukan registrasi secara mandiri melalui platform digital.

Komponen Utama Proyek
Proyek ini dibangun menggunakan struktur web standar (HTML, CSS, dan JavaScript) dengan pembagian fungsi sebagai berikut:

Halaman Utama (index.html): Berfungsi sebagai beranda yang menyambut calon siswa dengan informasi umum dan navigasi utama.

Halaman Alur (alur.html): Memberikan penjelasan visual atau prosedural mengenai langkah-langkah yang harus dilalui oleh pendaftar (mulai dari berkas hingga pengumuman).

Halaman Formulir (form.html): Inti dari sistem pendaftaran, tempat calon siswa mengisi data diri dan dokumen yang diperlukan.

Desain & Interaktivitas (style.css & script.js): CSS digunakan untuk memberikan tampilan yang profesional dan bersih (seperti yang terlihat pada gambar sebelumnya), sementara JavaScript menangani logika formulir dan navigasi.
